# Weekly NFL Game Picker

![Development Status](https://img.shields.io/badge/status-in%20development-yellow)

A web application for viewing and picking weekly NFL games, displaying live scores and team statistics powered by ESPN's API.

## Features

- Real-time NFL game scores and schedules
- Interactive team selection interface
- Color-coded team buttons with official NFL team colors
- Responsive design for desktop and mobile devices

## Tech Stack

- **Vanilla JavaScript** - Core application logic
- **Vite** - Build tool and development server
- **ESPN API** - Live NFL scores and game data
- **CSS3** - Styling and team color themes

## Live Demo

Access the application at: **[https://lab.carterirish.net/WeeklyGamePicker/](https://lab.carterirish.net/WeeklyGamePicker/)**

## API Reference

This project uses the ESPN NFL Scoreboard API:
```
https://site.api.espn.com/apis/site/v2/sports/football/nfl/scoreboard
```
